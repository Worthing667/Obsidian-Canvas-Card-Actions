# 徽章功能架构与服务组织

## 服务分层

### BadgeService（徽章服务）
负责徽章的业务逻辑处理

#### 方法：
- `setBadge(node, badge)` - 设置卡片徽章
- `getBadge(node)` - 获取卡片徽章
- `removeBadge(node)` - 移除卡片徽章
- `getBadgeType(content)` - 获取徽章类型

### BadgeModal（徽章模态框）
独立的模态框组件，负责徽章输入界面

#### 方法：
- `constructor(badgeService, node, currentBadge)` - 构造函数
- `onOpen()` - 打开模态框时的初始化
- `onSubmit(badgeText)` - 提交徽章内容
- `onClose()` - 关闭模态框时的清理

### BadgeStyleManager（徽章样式管理器）
负责徽章样式的注入和管理

#### 方法：
- `injectStyles()` - 注入徽章样式
- `removeStyles()` - 移除徽章样式

### OpenBadgeModalCommand（徽章命令）
处理徽章模态框的打开

#### 方法：
- `canExecute(context)` - 检查命令是否可执行
- `execute(context)` - 执行命令
- `getMenuTitle()` - 获取菜单标题

## 函数调用关系图

```
onload()
├── 注册右键菜单事件监听器
│   ├── canvas:node-menu 事件
│   │   └── addBadgeCommand(menu, node)
│   │       └── new BadgeModal(plugin, node, currentBadge).open()
│   │           └── getCurrentBadge(node)
│   └── canvas:selection-menu 事件
│       └── addBadgeCommand(menu, node)
│           └── new BadgeModal(plugin, node, currentBadge).open()
│               └── getCurrentBadge(node)
├── 注册文件变化监听器
│   └── metadataCache.on("changed")
│       └── updateNodeBadgeDisplay(node)
└── 注册布局就绪回调
    └── initializeAllCanvasBadges()
        └── updateNodeBadgeDisplay(node)

BadgeModal.onOpen()
└── 创建界面元素
    └── 点击"确定"按钮
        └── setBadge(badgeText)
            ├── 直接操作 node.nodeEl 属性
            └── 显示通知

getCurrentBadge(node)
└── 从 node.nodeEl.getAttribute("data-badge") 获取徽章内容

updateNodeBadgeDisplay(node)
├── 检查设置是否启用徽章功能
├── 如果未启用，移除徽章显示属性
└── 如果启用，保持现有属性（徽章信息已直接存储在属性中）
```

## 事件流

### 1. 右键菜单触发
```
用户右键点击卡片
→ 触发 canvas:node-menu 或 canvas:selection-menu 事件
→ CommandRegistry 处理命令路由
→ OpenBadgeModalCommand.execute()
→ 创建并显示 BadgeModal
```

### 2. 徽章设置
```
用户在模态框中输入内容并点击确定
→ BadgeModal.onSubmit()
→ BadgeService.setBadge(node, badge)
→ StorageAdapter 持久化存储
→ CSS 自动应用样式显示徽章
```

### 3. 初始化和更新
```
插件加载完成
→ main.ts onload() 中初始化所有服务
→ BadgeStyleManager.injectStyles()
→ 遍历所有 Canvas 节点并应用徽章样式
```

## 数据流

### 徽章数据存储
```
用户输入徽章内容
→ BadgeService.setBadge() 方法
→ StorageAdapter 将数据存储到 .canvas 文件
→ 同时设置 DOM 属性用于样式显示
→ CSS 选择器匹配并显示
```

### 徽章数据读取
```
显示模态框时需要预填充内容
→ BadgeService.getBadge() 方法
→ 从 .canvas 文件中读取持久化数据
→ 返回当前徽章内容
```

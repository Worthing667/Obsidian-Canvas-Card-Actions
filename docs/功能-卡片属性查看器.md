# 功能：卡片属性查看器

## 简介

卡片属性查看器是一个新的功能，允许用户查看和管理选中卡片的详细属性信息。该功能提供了直观的界面来查看卡片的尺寸、位置等信息，并支持批量操作来统一卡片尺寸。

## 功能特性

### 1. 属性查看
- 显示选中卡片的数量统计
- 展示卡片的尺寸范围、平均尺寸和位置范围
- 以表格形式列出每个卡片的详细信息，包括文本预览、尺寸、坐标位置和徽章内容

### 2. 批量操作
- 统一为最小尺寸：将所有选中卡片调整为最小的宽度和高度
- 统一为最大尺寸：将所有选中卡片调整为最大的宽度和高度
- 统一为平均尺寸：将所有选中卡片调整为平均宽度和高度
- 自定义尺寸：允许用户输入特定的宽度和高度来统一所有卡片

### 3. 单卡片操作
- 应用此尺寸：将某个卡片的尺寸应用到其他所有选中的卡片

### 4. 快捷操作
- 复制卡片尺寸：快速复制所有选中卡片的尺寸信息到剪贴板

## 使用方法

### 1. 访问属性查看器
1. 在Canvas中选择一个或多个文本卡片
2. 右键点击打开上下文菜单
3. 选择"查看卡片属性..."选项

### 2. 查看属性信息
- 属性查看器窗口将显示所有选中卡片的统计信息和详细列表
- 统计信息区域显示选中卡片的数量、尺寸范围、平均尺寸和位置范围
- 详细列表部分以表格形式展示每个卡片的具体信息

### 3. 批量操作
1. 在属性查看器底部的批量操作区域，您可以：
   - 点击"统一为最小尺寸"按钮将所有卡片调整为最小尺寸
   - 点击"统一为最大尺寸"按钮将所有卡片调整为最大尺寸
   - 点击"统一为平均尺寸"按钮将所有卡片调整为平均尺寸
   - 在自定义尺寸输入区域输入宽度和高度，然后点击"应用"按钮

### 4. 单卡片操作
1. 在详细列表中找到您想要作为模板的卡片
2. 点击该卡片行的"应用此尺寸"按钮
3. 其他所有选中的卡片将调整为该卡片的尺寸

### 5. 快捷操作
1. 在Canvas中选择一个或多个文本卡片
2. 右键点击打开上下文菜单
3. 选择"复制卡片尺寸"选项
4. 卡片尺寸信息将被复制到剪贴板

### 6. 快捷键
- 查看选中卡片的属性：在Canvas视图中选中卡片后使用快捷键
- 复制选中卡片的尺寸：在Canvas视图中选中卡片后使用快捷键

## 技术实现

### 文件结构
- `src/presentation/modals/CardPropertiesModal.ts`：实现属性查看器模态框
- `src/presentation/commands/PropertiesCommands.ts`：实现属性相关命令
- `src/main.ts`：集成新功能到右键菜单和快捷键系统

### 核心方法
- `analyzeCardSizes`：分析选中卡片的尺寸信息
- `unifyCardSizes`：统一卡片尺寸
- `undoLastSizeOperation`：撤销上一次尺寸操作

## 优势
1. **精确控制**：可以照着某个理想卡片的尺寸来调整其他卡片
2. **批量操作**：大大提高工作效率，特别是在处理大量卡片时
3. **直观统计**：快速了解选中卡片的整体情况
4. **撤销功能**：确保操作的安全性
5. **快捷操作**：提供快捷键支持，提高使用效率

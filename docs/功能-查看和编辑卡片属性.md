# 功能：查看和编辑卡片属性

## 概述

该功能为 Obsidian Canvas 提供统一的卡片属性管理入口。无论当前选中的是单张还是多张文本卡片，用户都通过同一个菜单项 `管理卡片属性` 进入，再由插件自动切换到最合适的属性查看器。

## 功能特点

### 1. 单卡片属性管理

- **信息显示**：展示当前卡片的尺寸、位置、徽章和内容预览
- **尺寸编辑**：支持独立修改宽度和高度
- **快捷复制**：支持复制尺寸、位置或完整属性

### 2. 多卡片属性管理

- **统计信息**：显示选中卡片数量、尺寸范围和位置范围
- **卡片列表**：汇总所有选中文本卡片的基础信息
- **批量调整**：支持统一尺寸和自定义尺寸
- **批量复制**：便于导出统计信息或尺寸信息

## 使用方法

### 单卡片操作

1. 在 Canvas 中右键点击单个文本卡片
2. 选择 `管理卡片属性`
3. 插件会打开单卡片属性查看器

### 多卡片操作

1. 在 Canvas 中选择多个文本卡片
2. 打开选择菜单
3. 选择 `管理卡片属性`
4. 插件会打开多卡片属性查看器

## 界面说明

### 单卡片属性查看器

- **卡片信息区域**：展示尺寸、位置、徽章和内容预览
- **尺寸编辑区域**：分别调整宽度和高度
- **复制功能区域**：复制尺寸信息、位置信息或完整属性

### 多卡片属性查看器

- **统计信息区域**：快速查看选中卡片整体情况
- **卡片列表区域**：逐项显示每张卡片的核心属性
- **批量操作区域**：统一宽高或应用自定义尺寸

## 相关命令

- `管理卡片属性`：主入口，自动根据选中数量切换查看器
- `复制选中卡片的尺寸`：独立命令，适合绑定快捷键或在命令面板中使用

## 技术实现

### 主要组件

1. `src/presentation/commands/PropertiesCommands.ts`
   - 提供统一的属性入口命令
   - 自动过滤非文本卡片
   - 根据选中数量决定打开单卡片或多卡片查看器

2. `src/presentation/modals/SingleCardPropertiesModal.ts`
   - 面向单张文本卡片
   - 专注于查看、复制和独立调整宽高

3. `src/presentation/modals/CardPropertiesModal.ts`
   - 面向多张文本卡片
   - 负责统计分析和批量尺寸操作

4. `src/services/CardService.ts`
   - 提供统一尺寸、统一宽度、统一高度等底层能力

### 数据处理流程

1. 从当前选择中筛出文本卡片
2. 根据数量打开单卡片或多卡片属性查看器
3. 用户执行复制或尺寸修改操作
4. 通过 `CardService` 和 Canvas API 更新节点数据
5. 调用保存逻辑将改动持久化到 `.canvas` 文件

## 适用场景

- 快速检查某张卡片的实际尺寸和位置
- 统一多个卡片的视觉规格
- 复制尺寸信息用于对齐其他卡片
- 批量整理画布中一组内容卡片的布局属性

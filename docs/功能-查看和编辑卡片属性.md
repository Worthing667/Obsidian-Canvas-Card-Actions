# 功能：查看和编辑卡片属性

## 概述
该功能允许用户查看和编辑 Obsidian Canvas 中卡片的各种属性，包括尺寸、位置等信息。通过专门的属性查看器，用户可以在一个统一的界面中完成所有相关的属性管理操作。

## 功能特点

### 1. 单卡片属性管理
- **信息显示**：显示卡片的当前尺寸、位置、徽章信息和内容预览
- **尺寸编辑**：支持独立调整卡片的宽度和高度
- **复制功能**：可复制尺寸信息、位置信息或完整属性

### 2. 多卡片属性管理
- **统计信息**：显示选中卡片的数量、尺寸范围、平均尺寸和位置范围
- **卡片列表**：以表格形式展示所有选中卡片的基本信息
- **批量操作**：支持统一为最小尺寸、最大尺寸或平均尺寸
- **自定义尺寸**：可为所有选中卡片应用自定义尺寸
- **批量复制**：可复制所有卡片的尺寸信息或统计信息

## 使用方法

### 1. 单卡片操作
1. 在 Canvas 中右键点击单个文本卡片
2. 选择"编辑卡片尺寸..."菜单项
3. 在弹出的属性查看器中进行相关操作

### 2. 多卡片操作
1. 在 Canvas 中选择多个文本卡片
2. 右键点击选中的卡片
3. 选择"批量管理卡片属性..."菜单项
4. 在弹出的属性查看器中进行相关操作

## 界面说明

### 单卡片属性查看器
- **卡片信息区域**：显示当前卡片的尺寸、位置、徽章和内容预览
- **尺寸编辑区域**：提供独立的宽度和高度编辑控件
- **复制功能区域**：提供三种复制选项（尺寸、位置、完整属性）

### 多卡片属性查看器
- **统计信息区域**：显示选中卡片的统计信息
- **卡片列表区域**：以表格形式展示所有选中卡片的基本信息
- **批量操作区域**：提供批量尺寸调整功能
- **复制功能区域**：提供批量复制选项

该功能采用模块化设计，主要包括以下组件：

1. **SingleCardPropertiesModal.ts** - 单卡片属性查看器
   - 专门处理单个卡片的属性查看和编辑
   - 提供简洁的界面，专注于核心功能
   - 集成复制功能，提升用户体验

2. **CardPropertiesModal.ts** - 多卡片属性查看器

   - 支持复杂的数据显示和操作

3. **PropertiesCommands.ts** - 属性查看命令处理器
   - 根据选中卡片数量选择合适的查看器

   - 集成依赖注入模式

### 核心功能实现

#### 尺寸验证和应用
```typescript
// 尺寸验证
private validateDimension(value: number): boolean {
  return !isNaN(value) && value >= 50 && value <= 2000;
}

// 应用单个维度尺寸
private async applySingleDimension(dimension: string, value: number): Promise<void> {
  try {
    const currentCard = this.cardInfos[0];
    const targetCard = this.cards[0];
    
    let newWidth: number, newHeight: number;
    
    if (dimension === "width") {
      newWidth = value;
      newHeight = currentCard.height; // 保持原有高度
    } else {
      newWidth = currentCard.width; // 保持原有宽度
      newHeight = value;
    }
    
    await this.cardService.unifyCardSizes([targetCard], { width: newWidth, height: newHeight });
    
    // 更新显示信息
    this.updateCurrentCardInfo(newWidth, newHeight);
  } catch (error) {
    console.error("更新卡片尺寸失败:", error);
    new Notice("更新尺寸失败: " + error.message);
  }
}
```

#### 剪贴板操作集成
```typescript
// 复制完整属性
const copyAllBtn = copyButtons.createEl("button", {
  text: "复制完整属性",
  cls: "copy-btn mod-cta"
});

copyAllBtn.addEventListener("click", async () => {
  let fullInfo = `卡片属性:
尺寸: ${this.cardData.width} × ${this.cardData.height} px
位置: X: ${this.cardData.x}, Y: ${this.cardData.y}`;

  if (this.cardData.badge) {
    fullInfo += `\n徽章: ${this.cardData.badge}`;
  }

  if (this.cardData.text) {
    fullInfo += `\n内容: ${this.cardData.text.substring(0, 100)}${this.cardData.text.length > 100 ? "..." : ""}`;
  }

  try {
    await navigator.clipboard.writeText(fullInfo);
    new Notice("完整属性信息已复制到剪贴板");
  } catch (error) {
    console.error("复制失败:", error);
    new Notice("复制失败，请重试");
  }
});
```

# 功能：按标记顺序复制内容

## 功能概述

“按标记顺序复制内容”允许用户根据卡片上的数字序号标记，对选中卡片内容进行排序和输出。这个能力适合整理任务步骤、章节结构和有层级的工作流。

## 使用方法

### 基本操作流程

1. 在 Canvas 中选中多张文本卡片
2. 通过 `一键复制` 或工作台把排序方式设为 `按标记顺序`
3. 输出当前顺序内容到剪贴板

### 输出格式

复制的内容格式为：

```text
[1] 卡片内容1

[2.1] 卡片内容2

[10] 卡片内容3
```

### 使用示例

假设 Canvas 中有以下卡片：

- 卡片 A：标记 `3`，内容“完成第三阶段测试”
- 卡片 B：标记 `1`，内容“制定项目计划”
- 卡片 C：无标记，内容“补充背景说明”
- 卡片 D：标记 `2.1`，内容“开发核心功能”

选中这些卡片后使用功能，复制到剪贴板的内容将是：

```text
[1] 制定项目计划

[2.1] 开发核心功能

[3] 完成第三阶段测试

补充背景说明
```

## 排序规则

- 有标记卡片优先，无标记卡片排在最后
- 标记之间使用 `localeCompare(b, undefined, { numeric: true })`
- 无标记卡片之间，继续复用当前位置排序规则
- 位置排序方向仍受 `设置卡片排序优先级` 影响

例如：

```text
1 < 2 < 2.1 < 2.10 < 10
```

## 技术特性

- 标记只保存字符串，不再区分数字 / 文字 / emoji 类型
- 选区快照会同时保留文本、位置和标记信息
- 按标记排序时不会再丢弃无标记卡片

## 适用场景

- 按步骤号导出任务清单
- 按章节号整理学习内容
- 在工作台中先按标记排序，再继续手动微调

## 注意事项

1. 标记只支持数字序号
2. 空白内容卡片仍会被忽略
3. 无标记卡片不会被过滤，而是排在最后
4. 标记模式下，有标记卡片会输出为 `[标记] 内容`

## 与其他功能的区别

| 功能 | 排序依据 | 输出格式 | 适用场景 |
|------|----------|----------|----------|
| 按位置复制 | 卡片在Canvas中的位置 | 纯文本内容 | 空间布局相关 |
| **按标记顺序复制** | **卡片的数字序号标记** | **[标记] 内容** | **编号、步骤、层级整理** |
| 普通复制 | 无排序 | 纯文本内容 | 简单内容获取 |

# 功能：卡片内容复制与排序

本插件为 Obsidian Canvas 提供了单卡片复制、多卡片按位置复制、按徽章复制和手动排序复制，覆盖从快速摘录到结构化导出的常见场景。

## 核心功能

### 1. 复制单张卡片内容

当您右键点击一张文本卡片时，可以使用菜单项 **`复制卡片内容`**。

- 会直接复制当前卡片的纯文本内容
- 适合快速引用某一张卡片的正文

### 2. 按位置复制内容

当您在 Canvas 中选中多张文本卡片时，可以使用 **`按位置复制内容`**。

- 插件会先根据卡片在画布中的空间位置进行排序
- 默认规则由设置中的排序优先级决定
- 每张卡片内容之间使用空行分隔

当前支持两种排序优先级：

- **优先按垂直方向排序**：从上到下，然后从左到右
- **优先按水平方向排序**：从左到右，然后从上到下

### 3. 按徽章顺序复制内容

当选中的多张卡片包含徽章时，可以使用 **`按徽章顺序复制内容`**。

- 插件会读取每张卡片的徽章内容
- 按徽章类型和内容排序后再复制
- 输出格式为 `[徽章] 内容`

这适合按优先级、状态或阶段导出内容。

### 4. 手动排序复制

当选中多张文本卡片时，还可以使用 **`手动排序复制`**。

- 会先按位置生成一个初始顺序
- 打开拖拽排序窗口后，您可以手动调整复制顺序
- 点击复制后，插件会按拖拽后的顺序输出纯文本内容

这个模式特别适合：

- 视觉布局并不完全等于最终输出顺序的场景
- 需要临时调整顺序但又不想移动卡片位置的场景
- 写作提纲、讲稿、演示流程等需要人工编排顺序的场景

## 使用方式

### 单卡片

1. 在 Canvas 中右键点击一张文本卡片
2. 选择 `复制卡片内容`

### 多卡片

1. 在 Canvas 中选中多张文本卡片
2. 打开选择菜单
3. 根据需要选择：
   - `按位置复制内容`
   - `按徽章顺序复制内容`
   - `手动排序复制`

## 使用场景

- **会议记录整理**：按画布顺序导出讨论要点
- **任务清单输出**：按徽章优先级整理待办事项
- **写作与演示编排**：通过手动排序复制组织最终内容顺序
- **头脑风暴归档**：将分散卡片整理成连续文本

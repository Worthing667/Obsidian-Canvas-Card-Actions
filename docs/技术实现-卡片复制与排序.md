# 技术实现：卡片复制与排序

本文档说明 `Canvas Card Actions` 当前版本中与卡片复制相关的实现细节。插件目前支持以下四种能力：

- 单卡片内容复制
- 按位置复制内容
- 按徽章顺序复制内容
- 手动排序复制

## 1. 菜单事件与选择数据获取

插件通过两个 Canvas 事件接入复制功能：

- `canvas:node-menu`：处理单卡片右键菜单
- `canvas:selection-menu`：处理多卡片选择菜单

当前实现的关键点是：`canvas:selection-menu` 的第二个参数是 `canvas` 实例，而不是选中项数组。真正的选择集合需要从 `canvas.selection` 读取。

```typescript
this.registerEvent(this.app.workspace.on("canvas:selection-menu", (menu: any, canvas: any) => {
    const selection = canvas.selection;

    if (!selection || selection.size === 0) {
        return;
    }

    this.setupCanvasServices(canvas);
    this.addSelectionMenuCommands(menu, selection);
}));
```

随后再统一转成数组：

```typescript
const selectionArray = Array.from(selection);
```

这样可以稳定拿到当前选中的 Canvas 节点。

## 2. 单卡片复制

单卡片复制由 `CopySingleCardCommand` 负责，菜单文案为 `复制卡片内容`。

```typescript
export class CopySingleCardCommand implements ICommand {
    async execute(): Promise<void> {
        await this.contentService.copySingleCardContent(this.node);
    }
}
```

底层逻辑会读取当前文本卡片内容并直接写入剪贴板。

## 3. 按位置复制内容

多卡片复制的基础模式是 `按位置复制内容`。

### 3.1 入口命令

```typescript
const copyByPositionCommand = new CopyByPositionCommand(
    this.contentService,
    selectionArray,
    this.settings.sortPriority
);
```

### 3.2 排序规则

排序优先级由设置项 `sortPriority` 控制：

- `yx`：优先按垂直方向排序
- `xy`：优先按水平方向排序

排序后会把文本内容用空行拼接，再写入剪贴板。

## 4. 按徽章顺序复制内容

`按徽章顺序复制内容` 会在排序前先读取每张卡片的徽章信息。

### 4.1 处理流程

1. 遍历选中的文本卡片
2. 调用徽章服务读取当前徽章
3. 根据徽章类型和内容排序
4. 生成 `[徽章] 内容` 格式的输出
5. 写入剪贴板

### 4.2 输出格式

```text
[1] 第一项内容

[2] 第二项内容
```

这让它非常适合优先级导出、状态整理和阶段性任务归档。

## 5. 手动排序复制

手动排序复制是当前复制能力里最灵活的一种。

### 5.1 命令入口

```typescript
const copyByManualOrderCommand = new CopyByManualOrderCommand(
    this.app,
    selectionArray
);
```

执行后会打开 `DragSortModal`。

### 5.2 初始顺序

`DragSortModal` 不会直接使用原始选择顺序，而是先按位置排序生成一个更自然的初始列表：

```typescript
const sorter = new PositionSortStrategy('yx', 10);
this.cardItems = sorter.sort(rawItems as unknown as any[]) as unknown as DragSortCardItem[];
```

这样用户一打开窗口，看到的顺序通常就已经接近最终输出顺序。

### 5.3 拖拽复制

在窗口中，用户可以通过拖拽重新排列卡片，最终按当前列表顺序复制纯文本内容：

```typescript
const content = this.cardItems.map((item) => item.text).join("\n\n");
await clipboardAdapter.writeTextWithNotice(content, `已按手动排序复制 ${this.cardItems.length} 张卡片的内容`);
```

## 6. 当前菜单结构

### 单卡片菜单

- `复制卡片内容`

### 多卡片选择菜单

- `按位置复制内容`
- `按徽章顺序复制内容`
- `手动排序复制`

## 7. 关键实现要点

### 7.1 统一过滤文本卡片

所有复制逻辑都只处理有效的文本卡片，避免文件卡片、网页卡片或空内容节点混入结果。

### 7.2 选择集与 Canvas 解耦

通过 `canvas.selection` 获取当前选中项，再转成数组传给命令对象，降低了命令层对 Canvas 事件结构的耦合。

### 7.3 复制体验分层

- 需要快速摘录时，用单卡片复制
- 需要自动排序时，用按位置复制
- 需要结构化编号时，用按徽章顺序复制
- 需要人工编排时，用手动排序复制
